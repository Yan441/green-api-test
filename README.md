# GREEN API Test Task

Test assignment for the position of DevOps Developer.

## Implemented methods

- getSettings
- getStateInstance
- sendMessage
- sendFileByUrl

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API

## Functionality

The page allows:
- entering `idInstance` and `apiTokenInstance`
- calling GREEN API methods
- displaying API responses in a read-only field
- sending text messages
- sending files by URL

## Repository

This repository contains a single-page HTML application for testing GREEN API methods.
